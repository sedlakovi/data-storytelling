Parametry scatter (platí nejen pro tuto metodu)¶

  • c - barva, nap. 'g', '#d24049', 'red' i seznam barev

  • alpha - prhlednost, 0 (upln prhledn) - 1 (neprhledn)

  • marker - tvar bodu, vce tady

  • s - velikost i seznam velikost

In [8]:
# Pomoci marker a s nastavime symbol a jeho velikost
fig, ax = plt.subplots(figsize=(10,6))
ax.scatter(x='Brainwt', y='Bodywt', data=data, c='#d24049', alpha=0.9, marker='s', s=200)
plt.show()

Přidat popisky¶

Pomoc ax.set meme nastavit limity a popisky os, nzev grafu, aj.

  • ax.set_xlim a ax.set_ylim nastav limit x a y osy

  • ax.set_xlabel, ax.set_ylabel nastav popisky x a y osy

In [11]:
fig, ax = plt.subplots(figsize=(10,6))
ax.scatter(x='Brainwt', y='Bodywt', data=data)
ax.set(
    xlim=[0, 1400], xlabel='Velikost mozku, g', ylabel='Velikost tla, kg',
    title='Velikost mozku a tla u primt'
)

plt.show()
In [12]:
fig, ax = plt.subplots(figsize=(10,6), dpi= 80)
ax.scatter(x='Brainwt', y='Bodywt', data=data)
ax.set(xlim=[0, 1400], xlabel='Velikost mozku, g', ylabel='Velikost tla, kg',
       title='Velikost mozku a tla u primt')
# iterrows iteruje pes dky tabulky
#  V kad iteraci udlme anotaci jednoho bodu grafu
for i, row in data.iterrows():
    ax.annotate(row.Species, (row.Brainwt, row.Bodywt))
